Graph Algorithm Visualizer

Python · NetworkX (or custom) · Matplotlib · Algorithms

This project lets users define graphs and visually step through algorithms like breadth-first search, depth-first search, and shortest path computation. It’s designed to help students and developers build intuition about how these algorithms explore a graph.

I focused on making the visualizations clear and the codebase easy to extend with new algorithms. The tool separates the core algorithm logic from the rendering layer, which makes testing and debugging easier.

Data Cleaning Toolkit

Python · Pandas · CLI

A small toolkit that automates common data-cleaning tasks: handling missing values, normalizing columns, detecting outliers, and generating quick summary reports. It is designed to be used from the command line so it fits easily into existing data workflows.

The focus was on reliability and transparency: each transformation is logged, and the reports make it clear what changes were applied to the data.
